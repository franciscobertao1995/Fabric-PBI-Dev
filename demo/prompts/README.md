# Prompt library

Copy-paste prompts for the [Contoso end-to-end demo](../README.md). Swap Contoso sales data for your own dataset as needed.

- [`01-agentic-scaffold.md`](01-agentic-scaffold.md) — plan and create the medallion lakehouses (Section 3).
- [`02-notebooks.md`](02-notebooks.md) — generate the Bronze/Silver/Gold notebooks (Section 2).
- [`03-model-optimize-document.md`](03-model-optimize-document.md) — optimize, annotate, and document the semantic model (Section 1).

> These prompts assume you have installed and configured the relevant tools (Skills for Fabric, the Fabric Data Engineering VS Code extension, and the Power BI Modeling MCP Server) as described in the [docs](../../docs/).
